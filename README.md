# CRC Error Detection Simulator

This project implements the Cyclic Redundancy Check (CRC) error detection technique using Python. The simulator allows users to enter binary data and a generator polynomial, generates the CRC remainder, creates the transmitted frame, and verifies received data for transmission errors. The project demonstrates how CRC is used in computer networks and communication systems to ensure data integrity and reliability.

## Features

* CRC code generation
* Error detection using modulo-2 division
* User-friendly console interface
* Sample test cases and outputs
* Educational implementation for learning networking concepts

## Technologies Used

* Python 3
* Git & GitHub

## Applications

* Computer Networks
* Data Communication
* Embedded Systems
* Digital Communication Systems
